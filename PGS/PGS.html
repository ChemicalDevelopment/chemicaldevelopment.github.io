Purpose

Prime Generator Search is a project written by Cade Brown (ChemicalDevelopment) whose main purpose is to find polynomials that generate primes for a specified length




Findings

Currently, the world record for most primes generated in a row is 80 (Hardy and Wright), which is the polynomial $(x - 40)^2 + (x - 40) + 41$. Only the first 40 are distinct, after that, the first 40 values are repeated.

The world record for most consecutive distinct primes in a row is held by Dress and Landreau (in 2002) and Gupta (2006). It is a 5th degree rational polynomial that generates 57 distinct primes in a row: $(n^5-133n^4+6729n^3-158379n^2+1720294n-6823316)/4$
